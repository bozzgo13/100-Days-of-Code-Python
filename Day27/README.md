# Miles to Kilometers Converter

A simple, lightweight **GUI application** built with Python and the **Tkinter** library that converts miles to kilometers in real-time.


## 🚀 Features
* **User-Friendly Interface:** Built with Tkinter for a native desktop feel.
* **Precise Calculation:** Uses the standard conversion factor ($1 \text{ mile} = 1.60934 \text{ km}$).
* **Rounding:** Automatically rounds the result to two decimal places for readability.
* **Grid Layout:** Organized UI elements using the `.grid()` geometry manager.

---

## 🖥️ How to Use
1.  **Run the script:** Execute the Python file in your terminal or IDE.
2.  **Input:** Enter the number of miles into the input field.
3.  **Calculate:** Click the **Calculate** button.
4.  **Result:** The equivalent distance in kilometers will appear instantly on the screen.

---

## 🛠️ Prerequisites
* **Python 3.x**
* **Tkinter** (usually comes pre-installed with Python on Windows and macOS).
